🍔 Foodhub Orders Dashboard
Welcome to my Foodhub Orders Analytics project!
This repository showcases how to analyze, visualize, and gain insights from Foodhub order data using SQL, Python, and Power BI. The goal is to understand customer behavior, restaurant performance, and delivery efficiency to drive business growth.

🚀 Project Overview
Objective: Analyze Foodhub orders to uncover trends in customer ratings, cuisine preferences, and delivery performance.

Tech Stack: SQL, Python (Pandas, Matplotlib, Seaborn), Power BI.

Output: Interactive Power BI dashboard with KPIs and visualizations.

📂 Repository Structure
data/ → Raw and cleaned Foodhub datasets.

sql/ → SQL scripts for data extraction and preprocessing.

python/ → Python notebooks for analysis and modeling.

dashboard/ → Power BI files and screenshots.

docs/ → Documentation and references.

📈 Key Features
Order Analysis: Track total orders, revenue, and ratings.

Segmentation: Break down by cuisine type and restaurant.

KPIs:

Order Pulse

Revenue Radar

Rating Review

Delivery Tracker

Cuisine Gauge

Visualizations:

Stacked Bar Chart → Cuisine Performance

Clustered Bar Chart → Restaurant Comparison

Pie Chart → Cuisine Share

Donut Chart → Revenue Share

Line Chart → Order Trend
